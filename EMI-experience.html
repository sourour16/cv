Lecturing Experience at EMI

Subjects taught

  • Project Management

    • Concepts of project management and approaches of selecting the adequate method of management Core knowledge of classic project management in line with PMP (Project Management Professional), Concepts of agile project management and applied Scrum in a real life project.

  • Object Oriented Project Management (Project course)

    • Course requires students to design and develop an application using a real case project management applying an object-oriented methodology of conception and development knowledge acquired in pre-requisite courses.

  • DOT NET web development

    • Introductory course to the dot net environment and first hand experience in developing web applications using the MVC model for programming.

  • Introduction to Database

    • Course introducing basic concepts of database management including the relational model, SQL queries and normalization of the relational scheme

  • Event-driven programming

    • Fundamentals of event-driven programming and application development programming using the Visual Basic.Net programming language: user interface design; creating object-oriented programs; accessing data; working with graphics.

  • Compilers (Project course)

    • Writing a basic compiler based on the major concepts and techniques used today in the implementation of programming language compilers including lexical analysis and parsing.

  • Embedded systems development

    • Practical exercises completing a course of embedded systems including conception and development of basic electronic circuits using Arduino and Raspberry electronic platforms

  • PC Hardware and software

    • Introduction to the functioning of a computer, ranging from Data processing to hardware components, basic networking and operating systems

  • Data structure

    • Advanced programming concepts focused on logical structures of data, analysis of algorithms and techniques leading to problem solving paradigms

  • Distributed programming practical exercises

    • Practical exercises completing a distributed programming course using the CORBA (Common Object Request Broker Architecture) Creating basic distributed client server applications, IDL language, inheritance and delegation approaches, naming service

Graduation projects supervision

Supervised graduation projects for master engineer students orientated on real projects management cases and sponsored by top leading companies.
